# NSS Public Keys

This directory contains public keys (`.nss` files) for the NSS Communicator project.

## NexaVision Admin Key

To send an encrypted message to the NexaVision team:

1. Import `nexavision-admin.nss` into your NSS Communicator extension
2. Compose a message and select "NexaVision Admin" as the recipient
3. Post or send the encrypted text anywhere — we'll decrypt it

**Request a key exchange:** Email your `.nss` public key file to **admin@nexavision.tech** and we'll add you as a contact.

## Adding Your Key

If you're a contributor or community member, you can submit a PR to add your public key here. Name it `yourname.nss`.

> ⚠️ **NEVER put private keys (`.ss` files) in this directory or anywhere in the repo.**
